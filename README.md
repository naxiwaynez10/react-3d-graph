### Compile and Run

Install the related NPM modules (required only during initial setup, or when package.json changes)
```bash 
yarn 
```

Install shadow-cljs globally:

```bash 
yarn
```

Compile and watch the CLJS
```bash 
shadow-cljs watch dev
```

Visit http://localhost:8060
